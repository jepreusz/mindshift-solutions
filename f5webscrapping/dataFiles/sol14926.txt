


TopicThis article applies to BIG-IP DNS (formerly BIG-IP GTM) 11.x through 12.x. For information about other versions, refer to the following article:SOL8140: Configuring the BIG-IP GTM system to accept underscore characters in wide IP names (10.x)You should consider using this procedure under the following condition:You want to use underscore ( _ ) characters in wide IP names.You want to use underscore ( _ ) characters in the BIG-IP GTM system's hostname.DescriptionBy default, the BIG-IP DNS system does not allow the use of underscore ( _ ) characters in wide IP names. Additionally, if the BIG-IP DNS system's hostname includes an underscore character, the ZoneRunner will fail to create DNS zones. However, you can configure the BIG-IP DNS system to accept underscore characters by using the Configuration utility or the Traffic Management Shell (tmsh).PrerequisitesYou must meet the following prerequisite to use this procedure:You have admin access to the Configuration utility or the tmshÂ utility.ProceduresTo configure the BIG-IP DNS system to accept underscore characters, perform one of the following procedures:Using the Configuration utility to configure the BIG-IP DNS system to accept underscore characters (BIG-IP 11.5.0 and later)Using the Configuration utility to configure the BIG-IP DNS system to accept underscore characters (BIG-IP 11.0.0 -
